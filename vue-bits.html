Vue of up to 16 bits

Little demo that shows the interaction between a text input field and checkboxes using the Vue.js framework. Depending on the value of the input field more or less checkboxes are rendered with a minimum of 4 and a maximum of 16. The minimum value of the input field is 0, the maximum is 131071.
